
# SQL Practice Solutions

## Overview

This repository contains my solutions to a set of SQL challenges provided in a public GitHub repository.

## Purpose

This project is part of my continuous learning as a data engineer, aiming to improve problem-solving skills and deepen my understanding of SQL, including:

* Data aggregation
* Window functions
* Ranking and ordering

## Source of the Exercises

The original questions can be found here:
*[(https://github.com/AlexanderConnelly/BrainTree_SQL_Coding_Challenge_Data_Analyst/tree/master)]*

## What I Did

I worked through each problem and wrote SQL queries to answer them using PostgreSQL.

Each solution focuses on:

* Writing clear and readable queries
* Using appropriate SQL features (e.g., `ROW_NUMBER()`, `SUM()`, `RANK()`)
* Ensuring correctness and efficiency

## Repository Structure

* The folder `scripts` → contains all the SQL queries (one file per question).
* The file `results.txt` → contains all the queries results.

## Key Concepts Covered

* Window functions
* Common Table Expressions (CTEs)
* Aggregations and grouping
* Subqueries
* Conditional logic in SQL
